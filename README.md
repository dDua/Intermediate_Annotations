drop_annotations.json

Contains 8500 annotations on random samples from DROP (https://arxiv.org/pdf/1903.00161.pdf)

Format:
```json
{
  "original_drop_query_id" : {
        [
          {
            "start_idx": "",
            "end_idx": "",
            "text": ""
          }
        ]
   }
}
```

```json
{

        [
          {
            "start_idx": "",
            "end_idx": "",

          }
        ]
   }
}
```
